계층형, 망형(Network), 관계형(Relational),

RDBMS

SQL문 (쿼리문) : 데이터베이스의 데이터를 직접 조작위한 언어

Java 프로그램 : java프로그래밍 언어로 만듦
데이터베이스 : SQL문 데이터베이스 생성, 삭제, 조회 가능

SQL문 : DBMS마다 조금씩 차이가 있음
기본문법은 동일

SQL문 종류
- DML(Data Manipulation Language) : 데이터 조작어
- DDL(Data Definition Language) : 데이터 정의어
- DCL(Data Control Language) : 데이터 제어어
- TCL(Transaction Control Language) : 트랜잭션 제어어

DML : insert(데이터 추가), delete(데이터 삭제),
update(데이터 수정), select(데이터베이스 데이터 조회)

DDL : CREATE(데이터베이스 테이블 생성),
ALTER(데이터베이스 테이블 수정),
DROP(데이터베이스 테이블 삭제),
RENAME(데이터베이스 테이블 이름 변경),
TRUNCATE(데이터베이스 테이블의 데이터 전체삭제)

DCL : GRANT(데이터베이스 사용자에게 권한 부여),
REVOKE(데이터베이스 사용자에게 권한 취소)

TCL : COMMIT(데이터베이스 변경내역 저장),
ROLLBACK(데이터베이스 변경내역 되돌리기=트랜잭션 변경내역 되돌리기)

데이터베이스 사용자 : 데이터베이스 사용하는 사람
트랜잭션(Transaction) : 데이터베이스 SQL문 실행단위
테이블 1개 생성 CREATE TABLE TEST_TABLE;
문장을 한번 실행시킨것을 트랜잭션 1개로 봄

PK(Primary Key) : 기본키를 뜻함, 테이블의 개별적인 데이터를 구분하기 위한 값(중복x)

FK(Foreign Key) : 외래키를 뜻함, 다른 테이블의 행을 식별할 수 있는 키를 의미

20231123

Table(테이블) : 데이터들을 모아놓은 집합
모아놓은 집합을 표 형태로 만들어놓은 것
공식적인 용어로는 릴레이션이라고 함

Tuple(튜플,투플) : 테이블에 존재하는 행들을 지칭함
(=Record) 데이터 리스트

Attribute(애트리뷰트, 어트리뷰트) : 테이블에 열 의미
(=Field, Column(칼럼))

Cardinality(카디널리티) : 튜플의 갯수를 의미

Degree(디그리) : 칼럼의 총 갯수

Domain(도메인) : 한 칼럼안에 나타낼 수 있는 값들의 집합

릴레이션 스키마 : 칼럼의 집합 (구조라고 하기도 함)
Ex)학번, 이름, 학과, 학년

릴레이션 인스턴스 : 실제값, 튜플의 집합


연관관계 종류
- 1:1 관계(일대일 관계)
한쪽 엔티티에서 상대 엔티티와 반드시 단 하나의 관계를 가지는 것
Ex) 남자 - 결혼 - 여자

- 1:N 관계(일대다 관계)
한쪽 엔티티가 상대 엔티티쪽에 여러 객체를 가질수 있음, 1:N 관계가 가장 많음, DB 설계시 가장 많이 쓰이는 관계

Ex) 부모 - 가족 - 자식
부모 : 1, 자식 : N
부모님 1 -> 자식 여러명

여러명의 자식 입장에서 부모 1명중
어떤 부모에 속해있는지 표현

부모테이블의 PK를 자식 테이블에 FK로 넣어서
관계를 표현

부모      자식
홍부모 - 홍길동

- N:M관계(다대다관계)

관계를 가진 양쪽 엔티티 모두에서
1:N 관계를 가지는 것을 의미

학원 - 수강 - 학생

한 학원 - 여러명의 학생 1:N
여러학원 - 한 학생 N:1

학원 PK -> 학생 FK
학생 PK -> 학원 FK

학원 PK, 학생PK를 컬럼으로 갖는
새로운 테이블을 만들어서 다대다 관계를 관리

정규화란

정규화(Normaliztion) :
테이블 간 중복된 데이터를 허용하지 않는다
데이터베이스 무결성 유지
(데이터의 정확성, 일관성 유지)

무결성(Integrity) : 
결함이 없는 성질(데이터 결함이 없는 상태)

데이터 결함?
-> 데이터 정확성, 일관성이 유지되지 않는
