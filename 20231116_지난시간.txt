- 제네릭(개념, 용어, 제네릭 클래스 사용법, 제한된 제네릭 클래스, 와일드카드 <?>, 제네릭 메서드)

제네릭 개념 : 입력받는 타입을 제한
제네릭 클래스 : class 클래스명<T>{}
제네릭 메서드 : static <T> void method1(){}
와일드 카드 : 타입변수 자리에 ? 기호 넣어줌

<? extends T> -> T와 T를 상속받는 모든 자손
<? super T> -> T와 T에게 상속해주는 모든 조상
<?> -> 모든 타입
